# Assets Directory

This directory contains static assets used by the video pipeline.

## Required Files

### bleep.mp3
A short bleep/censor sound effect for covering cuss words in audio.

**Specifications:**
- Format: MP3
- Duration: ~0.3-0.5 seconds
- Sample Rate: 44100 Hz

**Where to get:**
- [Freesound.org](https://freesound.org/search/?q=bleep+censor) (free, requires attribution)
- Generate using online tone generators (1000Hz tone)

Download a bleep sound and save it as `bleep.mp3` in this directory.
